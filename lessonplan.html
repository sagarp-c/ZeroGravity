Overview

This project aims to integrate satellite data, AI, and geospatial analytics for educational purposes, focusing on environmental science concepts.

Activities

  1. Introduction to Satellite Data:

    Begin with an overview of how satellite data is collected and its significance in monitoring the Earth's environment.

  2. API Exploration:

    Guide students to access and utilize satellite data through the provided APIs. Show them how to query satellite data and visualize it using basic tools.

  3. Group Project:

    Divide students into groups and assign them a project to analyze a specific environmental issue using satellite data. They will present their findings using visual data representations.
